# OSPFv3 IPv4 Address Family

With the FRR version currently used by this repository (`10.6.1`), `ospf6d` does not accept `address-family ipv4 unicast` under `router ospf6`, so a working OSPFv3-for-IPv4 sample cannot be provided here yet. The missing feature is tracked upstream as FRRouting issue [#6800 Support of Address Families in OSPFv3 (RFC 5838)](https://github.com/FRRouting/frr/issues/6800).
